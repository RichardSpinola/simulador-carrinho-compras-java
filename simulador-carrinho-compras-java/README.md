# Simulador de Carrinho de Compras (Java)

Simulador de carrinho de compras em Java, desenvolvido como atividade em dupla da disciplina de LPA (1º semestre — ADS, UCSAL).

## Tecnologias

- Java (uso de `Scanner` para entrada de dados via console)

## Funcionalidades

- Exibir catálogo de produtos (ID, nome, valor e quantidade em estoque)
- Adicionar produtos ao carrinho, com controle de estoque e limite de 20 itens
- Exibir os itens atualmente no carrinho
- Editar/remover quantidades já adicionadas ao carrinho
- Finalizar a compra, com opção de aplicar cupom de desconto (20%) e cálculo do valor total

## Como executar

```bash
javac src/CarrinhoDeCompras_Equipe6.java
java -cp src CarrinhoDeCompras_Equipe6
```
